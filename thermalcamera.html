Thermal Camera

Using the libraries created from the tutorial we will make a thermal camera based on the flir lepton sensor. It will feature a 240 x 320 4 wire resistive touch screen and mini sdcard slot for saving images.

Check youtube for a sneak peak. This was done on an Xmega that was overclocked to 40mhz. I will be working to convert this to ARM and also create a board that you can order from Dirtypcbs.com
